This folder is a multiclass model to predict the music's genre using spectrograph values
The spectrograph value were provided by https://www.kaggle.com/datasets/andradaolteanu/gtzan-dataset-music-genre-classification

When you run this python file, it outputs training loss and Accuracy. Looking at the data, its looks like it is a victim of overfitting, since the training loss is quite small. Validation set could have been to be 100% sure.
