This folder contains the first ML project I did: Binary Classification
When you run this python file, it reads the CSV inside the folder, run it through the model, then outputs loss, and accuracy of the model

This was to dip my toes into how these models work
Its quite simple to do
1) Extract data to use
2) Define your test, and training sets
3) Make the model. (Torch has a premade nn)
4) Make optimizer, and set learning rate and number of loops
5) Train model, then test it
